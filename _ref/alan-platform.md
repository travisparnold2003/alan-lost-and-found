## Travis's Project: Schiphol Lost & Found

**What it does:** A system to:

- Report a lost item (description, location, date lost, contact)
- Report a found item (description, location, date found, photo/contact)
- Browse/search lost and found items
- Match a lost item with a found item
- Mark items as resolved/claimed

**Why it's a good demo project:**

- Real-world use case (not a todo app)
- Covers CRUD, matching logic, state transitions
- Demonstrates understanding of Alan's patterns
- Useful enough that Rick can see real thinking went into it
